# BookApi
BookApi este o aplicație **WEB REST** dezvoltată în **C# (.NET 8)** care oferă operații **CRUD** pentru gestionarea unei liste de cărți.   Datele sunt stocate într-o bază de date locală **SQLite**, accesată prin **Entity Framework Core**.   Aplicația este containerizată și poate rula în **Docker**, conform cerințelor de laborator.
